# 📦 Move Box - 空間邏輯與色塊挑戰的完美結合

[English](README.md)

**Move Box** 是一款極具挑戰性的 3D 空間邏輯解謎遊戲。玩家需要操控一顆帶有數值的方塊，在精巧設計的網格地圖中滾動，目標是將正確數值與顏色的方塊精準推入目標點。

---

## 🚀 核心玩法
- **數值滾動**：方塊的每一面代表不同的數值（1-6），每滾動一格數值都會隨之改變。
- **色彩配對**：地圖中包含「變色器 (Changer)」，玩家必須策略性地規劃路徑，先改變方塊顏色，再將其推入對應顏色的目標點。
- **步數挑戰**：每一關都是對邏輯的終極考驗，嘗試用最少的步數達成目標！

---

## ✨ 亮點功能

### 1. 🌍 全球挑戰模式 (Challenge Mode)
- **雲端同步**：隨時從 GitHub 同步最新的挑戰關卡。
- **離線遊玩**：具備智慧快取機制，同步後即便在無網路環境下也能流暢遊玩。
- **進度記憶**：自動紀錄最後遊玩的關卡，進入選單即自動導航至當前進度。

### 2. 🛠️ 強大關卡編輯器 (Level Editor)
- **自由創作**：從地圖尺寸、方塊佈置到目標設定，全部由你決定。
- **模態安全操作**：內建防誤觸遮罩與清除確認機制，保護你的創作不遺失。
- **即時分享**：支援 JSON 格式匯入/匯出剪貼簿，輕鬆將你的燒腦設計分享給全世界！

### 3. 🔍 高效檔案管理
- **智慧搜尋**：在眾多自定義關卡中，透過關鍵字秒速找回你的檔案。
- **日期排序**：預設顯示最新編輯的關卡，專為創作者設計的優化體驗。
- **流暢捲動**：針對手機觸控優化的長列表操作，支援滑鼠拖曳與防誤觸偵測。

---

## 🎮 遊戲模式
- **Random 隨機模式**：系統自動生成無窮無盡的隨機謎題，永遠有新挑戰。
- **Challenge 挑戰模式**：攻略由開發者精心設計的經典關卡。
- **Custom 自定義模式**：遊玩你親手製作或朋友分享的精彩地圖。
