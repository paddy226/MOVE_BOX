# 📦 Move Box - The Ultimate 3D Spatial Logic Puzzle

[繁體中文](readme_tw.md)

**Move Box** is a challenging 3D spatial logic puzzle game. Players control a numeric cube rolling through intricate grid-based maps. The goal is to maneuver the cube into target spots with the matching value and color.

---

## 🚀 Core Mechanics
- **Numeric Rolling**: Each face of the cube represents a value (1-6). The value on top changes as you roll across the grid.
- **Color Matching**: Maps feature "Changers." You must strategically plan your path to change the cube's color before pushing it into the corresponding target.
- **Step Challenge**: Every level is an ultimate test of logic. Try to reach the goal in the fewest steps possible!

## ✨ Feature Highlights

### 1. 🌍 Global Challenge Mode
- **Cloud Sync**: Synchronize the latest challenge levels directly from GitHub.
- **Offline Play**: Features an intelligent caching engine that lets you play synced levels even without an internet connection.
- **Progress Memory**: Automatically tracks your last played level and navigates you back to your current progress instantly.

### 2. 🛠️ Powerful Level Editor
- **Full Creativity**: You decide everything from map size and tile layout to goal requirements.
- **Modal Safety UI**: Built-in input shields and clear-grid confirmations to protect your work from accidental touches.
- **Instant Sharing**: Support for JSON import/export via clipboard—share your brain-burning designs with the world easily!

### 3. 🔍 Advanced File Management
- **Smart Search**: Quickly find your custom levels using keywords in seconds.
- **Date Sorting**: Displays your most recently edited levels at the top—optimized for creators.
- **Smooth Scrolling**: Mobile-optimized list navigation with mouse-drag support and anti-accidental-touch detection.
