# Move Box 專案開發紀錄 (2026-03-17) 第十七次更新

## 1. 今日更新摘要
- **編輯器：關卡分享與導入系統 (Share & Import System)**：
    - **剪貼簿導入 (Import from Clipboard)**：新增導入按鈕（青色，圖示 `smaller.png`），支援直接從剪貼簿讀取 JSON 並生成關卡，方便社群交流。
    - **剪貼簿分享 (Export to Clipboard)**：優化分享按鈕（白色，圖示 `share2.png`），將關卡資料以標準 JSON 格式複製到剪貼簿。
    - **健壯性檢查**：實作了 JSON 解析驗證與資料結構檢查，確保無效資料不會導致程式崩潰，並提供清楚的 UI 錯誤提示。
- **UI 與圖示優化**：
    - 為編輯器工具列新增了專屬圖示，區分「本機載入」與「剪貼簿導入」，提升操作直覺性。
    - 新增了 `_show_alert` 全域彈窗輔助函式，統一編輯器內的提示風格。

## 2. 當前版本資訊
- **版本號**：v1.0.17

- **更新重點**：新增剪貼簿導入功能、分享圖示優化、編輯器健壯性提升。
