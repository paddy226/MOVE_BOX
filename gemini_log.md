# Move Box 專案開發紀錄 (2026-03-14) 第三次更新

## 1. 今日更新摘要
- **螢幕適配與視覺優化**：
    - **全螢幕適配 (Display Stretch)**：將 Godot 的 `stretch/aspect` 設定從 `keep` 改為 **`expand`**。現在 APK 在任何解析度（如 20:9 或 21:9）的 Android 手機上都能填滿整個螢幕，徹底消除了上下黑邊問題。
    - **UI 動態對位**：驗證了 HUD 各元件（步數、版本號、重新開始按鈕）的錨點設定。由於使用了 `set_anchors_and_offsets_preset` 搭配 `GROW_DIRECTION`，所有 UI 都會隨著螢幕變長而自動貼齊新的邊界，不會發生遮擋或位移。
- **穩定性提升**：
    - 重新整理了 `project.godot` 的視窗設定，確保在不同平台上都能有一致的縮放表現。

## 2. 當前版本資訊
- **版本號**：v1.0.5
- **作者**：Paddyliu
- **更新重點**：消除 Android 黑邊，全螢幕顯示適配。
