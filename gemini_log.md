# Move Box 專案開發紀錄 (2026-03-14) 第五次更新

## 1. 今日更新摘要
- **遊戲架構擴充與選單系統**：
    - **起始頁面 (Main Menu)**：建立了全新的 `menu.tscn` 作為遊戲入口。提供了 "Random" (隨機關卡)、"Custom" (自訂關卡) 與 "Editor" (關卡編輯器) 三個按鈕。
    - **JSON 關卡讀取**：實作了 `level.gd` 的 JSON 解析邏輯。現在可以從 `res://levels/` 目錄下載入獨立的關卡檔案，並精確生成地圖佈局。
    - **遊戲模式切換**：在 `GameState` 中新增了 `GameMode` 列舉，讓系統能根據玩家在選單的選擇（隨機或自訂）來切換場景邏輯。
- **介面優化 (UI Enhancement)**：
    - **返回按鈕 (Back Button)**：在遊戲場景左下角新增了左箭頭圖示按鈕，讓玩家隨時能退回主選單。
    - **國際化語言**：將選單按鈕改為簡潔的英文（Random, Custom, Editor），提升介面質感。
    - **檔案集中管理**：建立了 `levels/` 資料夾，統一存放 JSON 關卡檔案。

## 2. 當前版本資訊
- **版本號**：v1.0.7
- **作者**：Paddyliu
- **更新重點**：新增主選單、JSON 關卡讀取、返回功能。

## 3. 下一步計畫
1. 實作 **關卡編輯器 (Level Editor)**：允許玩家透過點擊直接編輯地圖並儲存為 JSON。
2. 實作 **關卡清單選擇器**：讓玩家可以在選單中看到並選擇不同的自訂關卡檔案。
