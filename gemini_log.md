# Move Box 專案開發紀錄 (2026-03-14) 第八次更新

## 1. 今日更新摘要
- **關卡編輯器完全體 (Level Editor Pro)**：
    - **一鍵預覽與無縫返回**：實作了 **PLAY** 按鈕。玩家可即時試玩設計中的關卡，且點擊「返回」後會自動帶回編輯器並 **完整恢復地圖進度**，大大提升開發效率。
    - **進階存取系統**：
        - **自訂檔名儲存**：新增儲存彈窗，支援自訂關卡名稱。
        - **智慧預填**：載入舊檔後點擊 SAVE 會自動預填原檔名，方便快速覆蓋或另存新檔。
        - **內建載入清單**：編輯器內即可開啟 **LOAD** 面板，瀏覽並切換不同關卡。
    - **安全刪除機制**：
        - **刪除確認彈窗**：點擊紅色 "X" 會彈出確認視窗，防止誤刪辛苦設計的關卡。
        - **權限區分**：PC 平台上允許刪除所有關卡（含內建），Android 上則保護內建關卡為唯讀。
    - **檔案管理優化**：關卡列表現在支援 **字母排序 (A-Z)**。
- **行動裝置體驗優化 (Mobile UX)**：
    - **UI 全面縮放**：針對手機操作，大幅加大了所有彈窗（SAVE, LOAD, SETTINGS, DELETE）的字體大小（32px+）與按鈕點擊區域。
    - **編輯器視覺增強**：
        - **孔洞預覽 (HOLE Preview)**：編輯器中孔洞以紅色半透明薄片顯示，方便辨識與編輯，但在正式遊戲中會完全消失呈現真實空位。
        - **操作鎖定**：彈窗開啟時自動禁用背景編輯與鏡頭平移，避免操作衝突。
