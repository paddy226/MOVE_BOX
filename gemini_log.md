# Move Box 專案開發紀錄 (2026-03-11)

## 1. 今日更新摘要
- **畫面調整**：將遊戲設定為 **直式 (Portrait)** 模式 (720x1280)，並優化了 UI 佈局。
- **圖示更新**：使用自製的 `resize_icon.bat` 工具，成功將 `return.png` 處理成符合 Android 規範的 Adaptive Icon (`return-s.png`)。
- **操控優化 (Android & PC)**：
    - **WASD / 方向鍵**：改為事件驅動模式，並過濾 Echo 事件，徹底解決 Android 鍵盤操作卡頓問題，並保留 PC 連續行走功能。
    - **鏡頭旋轉 (Q/E)**：改為 **30 度步進式旋轉**，搭配 Tween 平滑動畫，解決原本在行動裝置上輪詢不流暢的問題。
- **鏡頭功能**：
    - **復位功能**：增加 **右鍵雙擊**、**鍵盤 R 鍵**、以及 Android **兩指連點** 觸發鏡頭平滑復位至預設角度 (60度) 與距離 (7.0)。
- **UI 強化**：
    - 在右下角動態產生 **版本號顯示 (v1.0.1 by Paddyliu)**，並修正了長文字向右溢出的問題（改為向左生長）。
    - **換色點顯示**：優化換色點視覺，無限次數時不顯示數字，有限次數則正常顯示。

## 2. 核心技術變更
- **Input Handling**：將原本在 `_process` 的輪詢邏輯移至 `_unhandled_input`，提升輸入反應速度並降低效能消耗。
- **UI Positioning**：利用 `grow_horizontal = Control.GROW_DIRECTION_BEGIN` 確保 UI 元件在螢幕右側對齊的穩定性。
