# Move Box 專案開發紀錄 (2026-03-17) 第十六次更新

## 1. 今日更新摘要
- **成就系統：總步數統計 (Global Step Tracker)**：
    - **持久化儲存**：新增 `total_steps` 全域統計，並整合至 `challenge_progress.json`。玩家在所有模式下的每一次移動都會被納入統計且永久保存。
    - **階級稱號系統 (Rank System)**：根據總步數實作 10 倍數成長的階級系統 (Newbie, Walker, Runner, Master, Legend)，顯示於主選單。
- **UI 佈局絕對對稱化 (Perfect UI Symmetry)**：
    - **重構主選單**：改用統一的程式碼函式動態生成左右兩側的標籤。左側顯示「總步數與階級」，右側顯示「版本與作者」。
    - **精確對齊**：解決了 `.tscn` 讀取與動態建立標籤之間的高度、顏色與邊距差異，實現物理意義上的完全一致與對齊。
    - **視覺優化**：將底部資訊顏色調淡為淺灰色，減少視覺負擔，營造高級感。

## 2. 當前版本資訊
- **版本號**：v1.0.16
- **作者**：Paddyliu
- **更新重點**：新增總步數紀錄、階級系統、主選單 UI 對稱優化。
